# 📰 PRD: 출퇴근 뉴스레터 리더
**Product Requirements Document v0.5**
작성일: 2026-03-12 | 플랫폼: Web PWA (모바일 브라우저) | 배포: Vercel (무료)

---

## 1. 문제 정의 (Problem Statement)

> "영어는 알겠는데, 눈으로 읽으면 집중이 안 된다."

영어 독해 능력은 있지만 **긴 텍스트를 시각적으로 읽는 행위 자체**에서 집중력이 떨어지는 사용자가 있다. 특히 출퇴근 중 지하철/버스에서는 흔들림, 소음, 피로 등 환경적 방해요소가 더해진다. 기존 Gmail 앱으로 뉴스레터를 읽는 방식은 이 문제를 전혀 해결하지 못한다.

---

## 2. 타겟 사용자

- 영어 독해력은 있지만 **읽기 집중력**이 낮은 직장인
- 영어 뉴스레터를 구독하지만 **쌓아두고 못 읽는** 사람
- 출퇴근 시간(20~60분)을 활용해 **영어 인풋을 늘리고 싶은** 사람
- 주요 기기: **스마트폰 브라우저 (iOS Safari / Android Chrome)**

---

## 3. 핵심 인사이트

| 문제 | 원인 | 해결 방향 |
|------|------|-----------|
| 영어 읽기 집중 안 됨 | 시각 처리 부담 | **TTS로 들으면서** 텍스트 따라가기 |
| 긴 글이 버거움 | 전체 분량이 보임 | **문단 하나씩** 카드 형태로 분리 |
| 어려운 개념 이해 안 됨 | 배경지식 없음 | **다국어 AI 해설** 즉시 제공 |
| 출퇴근 중 사용 불편 | 화면 작고, 손 하나 | **스와이프 + 큰 버튼** UX |
| 읽다가 그만두게 됨 | 완독 동기 없음 | **진행률 + 완독 기록** |

## 4. 핵심 기능 (Features)

### 🔴 Must Have (MVP)

#### F1. 문단 카드 리더
- 뉴스레터를 문단 단위로 잘라 **한 번에 하나씩** 보여줌
- 스와이프 좌우로 이전/다음 이동
- 전체 진행률 프로그레스 바 표시

#### F2. TTS 읽어주기 (핵심)
- 문단 텍스트를 **미국 영어 음성으로 읽어줌**
- 읽히는 단어/문장 **하이라이트 싱크**
  - Web Speech API의 `SpeechSynthesisUtterance.onboundary` 이벤트로 **단어 단위 하이라이트** 가능
  - iOS Safari / Android Chrome 모두 지원 — 크로스 플랫폼 일관성 확보
- 재생 속도 조절 (0.8x ~ 1.5x) — `SpeechSynthesisUtterance.rate`
- **화면 꺼짐 시**: PWA는 백그라운드 TTS가 제한적 → 화면 켜진 상태에서 사용 (Wake Lock API로 화면 꺼짐 방지 가능)

#### F3. 다국어 AI 해설
- 버튼을 눌러야 해설 표시 (자동 표시 안 함)
- 사용자가 설정한 언어로 AI가 쉽게 설명 (한국어, 일본어, 스페인어 등)
- **다국어 지원은 프롬프트 변경만으로 가능** — 추가 플랜 불필요, 사용량 과금
- 전문 용어는 괄호로 풀어서 설명, 2~3문장으로 간결하게
- **API 키 방식 (MVP)**: 사용자가 직접 Claude/Gemini API 키를 입력 → 서버 불필요, 비용 사용자 부담
  - 설정 화면에서 API 키 입력 (localStorage 저장, 개인용이므로 충분)
  - 지원 모델: Claude Haiku (저렴), Gemini Flash (무료 티어 있음)
- **오프라인 시**: "인터넷 연결이 필요합니다" 안내 표시, 해설 버튼 비활성화

#### F4. 모바일 최적화 PWA UI
- 전체 화면 카드 UI (스크롤 없음)
- 최소 44px 터치 영역
- 한 손 조작 가능한 버튼 배치 (하단)
- 다크 모드 지원 (지하철 환경, `prefers-color-scheme` + 수동 토글)
- PWA manifest로 **홈 화면 추가** 시 네이티브 앱처럼 실행 (주소창 없음)

#### F5. Gmail 자동 연동 (MVP 필수)
- Google OAuth 로그인
- 받은편지함에서 뉴스레터 자동 감지
  - **감지 기준 (MVP)**: 발신자 화이트리스트 (Morning Brew, TLDR, Substack 등 주요 뉴스레터 사전 등록) + 사용자가 수동으로 발신자 추가 가능
  - **v1.1**: `List-Unsubscribe` 헤더 기반 자동 분류 추가
- 읽지 않은 뉴스레터 목록 표시
- **이메일 본문 파싱**: HTML → 텍스트 변환 필수 (뉴스레터는 거의 100% HTML). `htmlparser2` 또는 `cheerio` 라이브러리로 본문 추출, 이미지/광고 블록 제거
- **MVP에 포함하는 이유**: 수동 복붙 플로우는 마찰이 커서 습관 형성이 안 됨. 앱 열면 뉴스레터가 이미 와있어야 매일 쓰게 됨.
- **Google OAuth**: 개인용이므로 테스트 모드(본인 계정만 등록)로 충분. 앱 검증 불필요

#### F5-1. 텍스트 붙여넣기 (폴백)
- Gmail 연동 실패 또는 다른 소스(Substack 웹 등)에서 복사한 경우를 위한 보조 입력 수단

## 5. 사용자 여정 (User Journey)

```
[출근길 지하철 탑승]
        ↓
앱 열기 → 오늘 읽을 뉴스레터 선택 (or 붙여넣기)
        ↓
▶ 재생 버튼 → TTS가 첫 문단 읽어줌 (미국식 영어)
  + 텍스트 하이라이트 싱크
        ↓
이해 안 되면 → [해설 보기] 버튼 → 선택한 언어로 AI 설명
        ↓
스와이프 → 다음 문단
        ↓
[목적지 도착] → 진행률 자동 저장
        ↓
[퇴근길] → 이어서 읽기
        ↓
완독 🎉 → 읽기 기록 저장
```

## 6. 기술 스택

| 구분 | MVP | 비고 |
|------|-----|------|
| 프레임워크 | **Next.js (App Router)** | React 기반, PWA 지원, Vercel 무료 배포 |
| TTS | **Web Speech API** (브라우저 내장, 무료) | `onboundary`로 단어 하이라이트 싱크 |
| 화면 꺼짐 방지 | **Wake Lock API** | TTS 재생 중 화면 유지 |
| AI 해설 | Claude Haiku / Gemini Flash (사용자 API 키) | 클라이언트에서 직접 호출 |
| Gmail 연동 | **Gmail API (Google OAuth)** | 개인용 테스트 모드 |
| HTML 파싱 | `htmlparser2` (이메일 본문 → 텍스트) | 클라이언트 사이드 처리 |
| API 키 저장 | `localStorage` | 개인용이므로 충분 |
| 저장/기록 | `localStorage` / `IndexedDB` | 진행률, 설정 등 로컬 저장 |
| PWA | `next-pwa` + Web App Manifest | 홈 화면 추가 시 앱처럼 실행 |
| 스와이프 | `react-swipeable` 또는 터치 이벤트 직접 구현 | 카드 좌우 이동 |
| 배포 | **Vercel** (무료) | git push → 자동 배포 |

> **React Native → Next.js로 전환한 이유**:
> - PWA로 배포하면 **앱스토어/빌드 과정 전부 불필요**
> - Web Speech API가 `onboundary` 이벤트로 **단어 단위 하이라이트를 iOS/Android 모두** 일관되게 지원
> - Vercel 무료 배포로 **어떤 기기든 URL 접속만으로 사용** 가능
> - 개발 복잡도가 대폭 감소 (네이티브 모듈, 빌드 설정 등 불필요)

### 10-3. AI 해설 비용 (사용자 부담 — BYOK 방식)

MVP에서는 사용자가 직접 API 키를 입력하므로 **개발자 비용 $0**.

**사용자 관점 예상 비용** (문단 1개 해설 기준, input ~500 / output ~200 tokens):

| 모델 | 1회 비용 | 하루 10문단 | 월간 (22일) | 특징 |
|------|---------|-----------|-----------|------|
| **Gemini Flash** | $0 | $0 | **$0** | 무료 티어 (분당 15회, 일 1,500회) |
| Claude Haiku | $0.0012 | $0.012 | **~$0.26** | 가장 저렴한 Claude |
| Claude Sonnet | $0.0045 | $0.045 | **~$1.00** | 더 정교한 해설 |

**추천 전략**: Gemini Flash를 기본값으로, Claude Haiku를 프리미엄 옵션으로 제공.
→ 사용자가 원하면 $0으로도 AI 해설 사용 가능.
